# Otto Documentation

Documentation site for [Otto](https://github.com/ebsc-dev/otto) - AI-powered development automation.

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Structure

```
content/
└── docs/
    ├── 1.getting-started/    # Introduction, quickstart, concepts
    ├── 2.use-cases/          # Ticket-to-PR, PR feedback, clarification
    ├── 3.workflows/          # Linear, GitHub, agent configuration
    └── 4.billing/            # Overview, pricing, token usage
```

## Tech Stack

- [Nuxt 3](https://nuxt.com)
