# Pricing Tiers

Otto offers volume-based pricing — the more you use, the lower your effective rate.

## Tier Structure

| Tier | Monthly Usage | Multiplier | Effective Discount |
|------|---------------|------------|-------------------|
| **Tier 1** | < $500 | 2.5× | Base rate |
| **Tier 2** | $500 - $1,000 | 2.0× | 20% off |
| **Tier 3** | > $1,000 | 1.5× | 40% off |

## How Tiers Work

Your tier is determined by your **cumulative credit usage** within a calendar month.

### Example

1. **Month starts** — You're at Tier 1 (2.5× multiplier)
2. **Usage hits $500** — You move to Tier 2 (2.0× multiplier)
3. **Usage hits $1,000** — You move to Tier 3 (1.5× multiplier)
4. **Month resets** — Back to Tier 1

## Calculating Costs

### Base Token Costs

These are the underlying AI provider costs (approximate):

| Model | Input | Output |
|-------|-------|--------|
| Claude Sonnet | $3 / 1M tokens | $15 / 1M tokens |
| Claude Opus | $15 / 1M tokens | $75 / 1M tokens |
| Claude Haiku | $0.25 / 1M tokens | $1.25 / 1M tokens |

### With Multiplier

If you're in **Tier 1** and use Claude Sonnet:

```
Input: $3 × 2.5 = $7.50 / 1M tokens
Output: $15 × 2.5 = $37.50 / 1M tokens
```

In **Tier 3**:

```
Input: $3 × 1.5 = $4.50 / 1M tokens
Output: $15 × 1.5 = $22.50 / 1M tokens
```

## Maximizing Value

### Reach Higher Tiers Faster

- Front-load credits at the start of the month
- Run batch operations together

### Use Appropriate Models

- **Haiku** for simple tasks (clarification, small fixes)
- **Sonnet** for most implementations (best balance)
- **Opus** for complex architecture decisions

### Optimize Context

- Keep agent context concise but complete
- Avoid redundant information
- Use well-structured tickets

## FAQ

### Do unused credits roll over?

Yes, your credit balance carries over. Only the tier resets monthly.

### Can I upgrade mid-month?

Tiers are automatic based on usage — no manual upgrade needed.

### What if I exceed my balance?

Agents will pause until you add more credits. You'll receive a warning before this happens.
